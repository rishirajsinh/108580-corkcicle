# 🥤 Corkcicle Website Clone

A high-fidelity landing page and multi-page clone of the **Corkcicle** website. This project focuses on modern web design, responsive layouts, and clean CSS styling to mimic a premium e-commerce user experience.

## 🚀 Live Demo
*(If you have deployed this project, add the link here)*
🔗 [https://108580-corkcicle.netlify.app/]

## 🛠️ Tech Stack

- **HTML5**: Semantic structure for multiple pages.
- **CSS3**: Custom styling, Flexbox, and Grid for responsiveness.
- **Assets**: High-quality product images and iconography.

## ✨ Features

- **Responsive Design**: Optimized for different screen sizes.
- **Multi-page Navigation**: 
  - `index.html`: Main landing page with featured products.
  - `coffee.html`: Specific category page for coffee-related items.
  - `account.html`: User account/login interface.
  - `new.html`: Latest arrivals and new collections.
- **Premium UI**: Smooth hover effects, elegant typography, and clean spacing.
- **Interactive Elements**: Navigation bar, shopping bag icons, and user-friendly layouts.

## 📁 Project Structure

```text
108580-corkcicle/
├── index.html           # Home Page
├── coffee.html          # Coffee Category Page
├── account.html         # User Account Page
├── new.html             # New Arrivals Page

└── (images)             # Product and UI assets
```

## 🧑‍💻 How to Run Locally

1. **Clone the repository**:
   ```bash
   git clone https://github.com/rishirajsinh/108580-corkcicle.git
   ```

2. **Navigate to the project folder**:
   ```bash
   cd 108580-corkcicle
   ```

3. **Open in Browser**:
   Simply open `index.html` in any web browser.
